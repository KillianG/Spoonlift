# Screenshots

Drop PNGs here and they'll show up in the top-level README. Suggested shots:

| File name                | What to capture |
|--------------------------|------------------|
| `workspace-light.png`    | Full window, dual-pane, List view on the left / Icons on the right, sidebar visible |
| `workspace-dark.png`     | Same as above but in Dark mode |
| `columns-view.png`       | Columns view in a single pane showing 3+ columns deep |
| `context-menu.png`       | Right-click menu expanded on a selected item |
| `transfer.gif`           | Drag-drop a file between panes, showing the transfer progress window |
| `tags.png`               | A file with multiple Finder tag dots visible, tag submenu open |

**How to capture:**
- Full window: `⇧⌘4` then `Space`, click the window — produces a PNG with macOS's standard drop shadow.
- GIFs: use [Kap](https://getkap.co) (free, open source) or Gifski.
- Keep PNGs under ~800 KB; GIFs under ~3 MB. Resize to 1600px wide max.
